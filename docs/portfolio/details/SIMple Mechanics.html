SIMple Mechanics is one part of my SIMple Physics project. It is a simulator whose goal is to make it easier to learn and teach 2D mechanics without expensive lab equipment or in person demonstrations.

  • Uses an immediate mode GUI framework
  • In addition to real time graphics, outputs simulation data as a CSV file for further analysis
  • Includes highly integrated Lua scripting for custom scenarios and interactions
